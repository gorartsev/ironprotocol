# IRONPROTOCOL — Как добавить на iPhone

## Вариант 1: GitHub Pages (бесплатно, навсегда)

### Шаг 1: Создай репозиторий
1. Открой github.com на телефоне или компе
2. Нажми "+" → "New repository"
3. Назови: `ironprotocol`
4. Поставь галку "Public"
5. Нажми "Create repository"

### Шаг 2: Загрузи файлы
1. На странице пустого репо нажми "uploading an existing file"
2. Загрузи ВСЕ 4 файла из этой папки:
   - `index.html`
   - `manifest.json`
   - `icon-192.png`
   - `icon-512.png`
3. Нажми "Commit changes"

### Шаг 3: Включи GitHub Pages
1. Зайди в Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: "main", папка: "/ (root)"
4. Нажми Save
5. Через 1-2 минуты сайт будет доступен по адресу:
   `https://ТВОЙ_ЛОГИН.github.io/ironprotocol/`

### Шаг 4: Добавь на домашний экран iPhone
1. Открой ссылку в **Safari** (не Chrome!)
2. Нажми кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Прокрути вниз → "На экран «Домой»"
4. Назови "IRON" → нажми "Добавить"

Готово! Приложение будет открываться как нативное — без адресной строки Safari.

## Важно
- Все данные (прогресс, веса) хранятся ЛОКАЛЬНО на устройстве
- Работает без интернета после первого открытия
- Если очистишь данные Safari — прогресс сотрётся
